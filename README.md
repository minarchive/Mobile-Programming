# Mobile-Programming
Mobile Programming class of 2024 : O_to_I 

## About
 
**OTOI** stands for **Outsider to Insider** - an app designed for people who struggle to find the right words in conversation. OTOI recommends contextually appropriate replies based on your chat history and relationship settings, helping users communicate more naturally and confidently.

## Features
 
### 01. Answer Recommendation
The core feature of OTOI. Input your message and the app recommends 3 tailored replies based on:
- The relationship and attitude profile of the contact
- The actual chat history context
- The tone you want to convey
**Screens**
- **Chatroom list** - search contacts, add or delete chatrooms
- **Chatting screen** - switch between User/Agent roles, request answer recommendations, delete messages

### 02. Simulation
Practice real-world conversations with AI partners before the actual thing.
- Select an AI persona with a defined relationship and attitude
- Engage in lifelike simulated chat
- Roles are allocated to make the practice feel authentic
**Screens**
- **Simulation room list** - search, add, and delete simulation rooms
- **Simulation chat** - lifelike chat practice with the AI partner

### 03. Small Talk
Never run out of things to say. OTOI suggests conversation topics in two ways:
- **Basic recommendations** - pre-curated topic lists organized by category (first meeting, travel, food, movies, etc.)
- **LLM recommendations** - dynamically generated topic suggestions powered by an LLM

### 04. Additional Features
- **Tutorial** - onboarding walkthrough of all app features
- **Loading screen** - customizable character displayed during loading
- **Settings** - manage loading screen character, view personal information policy, contact us, and check app version
